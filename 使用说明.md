## 1. 快速开始

### 启动游戏

游戏是单 HTML 文件，**不需要任何安装**。有两种启动方式：

**方式一：本地服务器（推荐）**

```bash
cd /workspace
python3 -m http.server 8000
```

浏览器访问：`http://localhost:8000/tank-battle.html`

**方式二：直接打开**

双击 `tank-battle.html` 用浏览器打开即可（部分功能如本地存储、模块加载建议用方式一）。

### 系统要求

- 现代浏览器（Chrome / Edge / Firefox / Safari 最新版）
- 推荐分辨率 1280×720 及以上
- 支持触屏的设备自动启用虚拟摇杆
- 支持 Web Audio API（用于音效）

## 2. 基础操作

### 键盘控制

| 按键 | 功能 |
|------|------|

## 4. 战斗系统

### 战斗流程

1. **3-2-1 倒计时** - 每关开始前的准备
2. **消灭所有敌人** - 清完即可进入下一关
3. **关卡间商店** - 用金币购买增益
4. **每 5 关 Boss 战** - 击败 Boss 后进入下一阶段
5. **生命归零** - 弹出游戏结束界面

### 战斗机制

- **坦克移动** - 四个方向独立控制，可与射击同时进行（需解锁"移动射击"技能）
- **射击** - 单发子弹，遇到障碍或敌人会爆炸
- **碰撞** - 坦克之间、坦克与钢墙之间会阻挡移动
- **破坏地形** - 子弹可击碎砖墙、钢墙（重型敌人可破钢墙）
- **伤害飘字** - 命中时屏幕显示伤害数字，击杀显示金色得分
- **连击** - 短时间内连续击杀累计连击数，分数加成更高

### 死亡与复活

- 死亡时屏幕全屏红色闪烁（不再卡死，可正常进入结束界面）
- 在商店可购买"额外生命"道具
- "护盾"道具可使玩家免疫一次伤害

## 5. 关卡与敌人

### 关卡主题

关卡地图主题按关卡循环出现：

| 主题 | 出现关卡 | 视觉特色 |
|------|---------|---------|
| **城市** | 1, 5, 9... | 灰蓝色调 + 建筑物 |
| **沙漠** | 2, 6, 10... | 黄色调 + 仙人掌 |
| **雪地** | 3, 7, 11... | 白色调 + 雪花 |
| **夜晚** | 4, 8, 12... | 暗色调 + 灯光 |

### 敌人类型

| 类型 | 血量 | 得分 | 特性 |
|------|------|------|------|
| **基础坦克** | 1 | 100 | 标准移动与射击 |
| **快速坦克** | 1 | 200 | 移动速度 2 倍 |
| **重型坦克** | 3 | 300 | 多发子弹、可破钢墙 |
| **精英坦克** | 5 | 500 | 智能追踪玩家 |
| **Boss** | 50+ | 2000+ | 每 5 关出现，有专属技能 |

### Boss 战

- **触发条件**：每通关 5 关出现一次
- **血量**：至少 50
- **狂暴模式**：血量低于 50% 时切换
- **攻击方式**：
  - 扇形弹幕（5-7 发扇形）
  - 环形弹幕（全方向子弹）
  - 冲撞攻击（直线冲刺）

### 敌人编队

敌人生成时会有编队：
- **直线** - 一字排开前进
- **V 字** - 楔形包围
- **包围** - 从四面包围玩家

### 昼夜系统

每 2 分钟（游戏内）切换一次时间：

- **白天** - 正常视野
- **黄昏** - 略微变暗
- **夜晚** - 视野受限（仅坦克周围可见）
- **黎明** - 逐渐变亮

侧边栏的小地图会显示当前时间。

| **空袭** | 全场敌人受到伤害 | 即时 |
| **冰冻** | 冻结所有敌人 3 秒 | 即时 |

侧边栏道具栏上方会显示当前持有的道具图标，激活时高亮发光。

## 10. 地图编辑器

### 进入方式

主菜单 → **地图管理** → **新建地图**

### 编辑工具栏

左侧工具栏包含所有可绘制的地形：

| 工具 | 用途 |
|------|------|
| **空地** | 擦除地形 |
| **砖墙** | 子弹可击碎 |
| **钢墙** | 普通子弹无法破坏（重型可破） |
| **森林** | 装饰，可通过 |
| **水域** | 坦克无法通过，子弹可过 |
| **冰面** | 坦克打滑通过 |
| **油桶** | 击中或碰撞会爆炸 |
| **陷阱** | 踩到扣血 |
| **出生点（蓝）** | 蓝方玩家出生位置 |
| **出生点（红）** | 红方玩家出生位置 |
| **道具点** | 标记为道具刷新区 |

### 操作方式

- **左键点击格子** - 绘制当前选中工具
- **左键拖动** - 连续绘制
- **右键点击** - 快速擦除
- **底部按钮**：
  - **保存** - 输入地图名称保存到地图库
  - **随机生成** - 自动生成示例地图
  - **返回** - 放弃编辑

### 保存地图

1. 点击 **保存** 按钮
2. 弹出输入框（自定义输入框，全屏可用）
3. 输入地图名称（默认填入上次使用名）
4. 点击 **确定** 保存
5. 地图会出现在 **地图管理** 列表中，可在游戏中加载

### 地图管理

- **加载** - 将地图设为当前游戏关卡
- **删除** - 自定义确认对话框，避免误删
- **导入/导出** - 与其他玩家分享地图

## 12. 观战模式

## 18. 商店与金币

### 进入方式

### 商店物品

| 物品 | 价格 | 效果 |

## 20. 常见问题

### Q1: 进入全屏后弹出对话框会退出全屏？

**已修复。** 本游戏使用自定义的确认对话框（`showConfirm`）和输入对话框（`showPrompt`），**全屏模式下正常工作**，不会触发浏览器原生 `alert` / `confirm` / `prompt` 而退出全屏。

### Q2: 保存地图时报 `Error: prompt() is not supported`？

**已修复。** 所有原本使用原生 `prompt()` 的位置已替换为自定义输入对话框。保存地图时会弹出游戏内置的输入框，可正常输入名称。

### Q3: 死亡后红色屏幕卡死？

**已修复。** 死亡时全屏红色闪烁效果正常显示，游戏结束后会自动弹出结束界面，可正常返回主菜单。

### Q4: 观战模式没有坦克出现？

**已修复。** 观战模式现在会正常生成红蓝双方 AI 坦克，AI 行为与正常比赛一致（智能旋转、追踪、攻击）。

### Q5: 回放模式无法播放？

**已修复。** 主循环已支持回放模式，会按帧重现历史对战。

### Q6: 删除存档 / 地图 / 模组后无法回到对应界面？

**已修复。** 删除操作使用自定义确认对话框，删除完成后会自动跳回对应管理界面（存档管理 / 地图管理 / 模组管理）。

### Q7: 正常模式下地图渲染异常？

**已修复。** `startLevel` 会标记 `MapRenderer.needsRefresh = true`，关卡地图正常显示。

### Q8: 移动端如何操作？

游戏会自动检测触摸设备，显示虚拟摇杆（左）与射击按钮（右），无需额外设置。

### Q9: 数据保存在哪里？

所有数据保存在浏览器 `localStorage` 中，**清除浏览器数据会导致存档丢失**，建议定期使用 **存档管理 → 导出存档** 备份。

### Q10: 可以离线玩吗？

完全可以。游戏是单 HTML 文件，不需要服务器（仅在本地打开时可能因 file:// 协议限制部分功能，建议启动简单 HTTP 服务器）。
